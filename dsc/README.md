# dsc

Dyld shared cache utilities.  
Invoke `build.sh` with path to dyld source folder to build `dsc_extractor`, `dsc_util` and, for dyld-519 and later, `dsc_closure`.

### Version support

Verified to compile with:

- dyld-421.1
- dyld-421.2
- dyld-433.5
- dyld-519.2.1
- dyld-519.2.2
- dyld-551.3
- dyld-551.4
- dyld-625.13
- dyld-635.2
- dyld-640.2
- dyld-655.1
- dyld-655.1.1
- dyld-732.8
- dyld-733.6
- dyld-733.8
- dyld-750.5
- dyld-750.6
- dyld-832.7.1
